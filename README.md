# AT89C51 LED Control Circuit (Proteus Simulation)

This project demonstrates a basic LED control circuit designed using the **AT89C51 microcontroller** and simulated in **Proteus**.  
The aim of the project is to understand microcontroller interfacing, crystal oscillator configuration, and digital output control using LEDs.

## 📌 Project Overview

The circuit is built around the **AT89C51 microcontroller**, which controls multiple LEDs connected to its I/O ports.  
Each LED is connected through current-limiting resistors to ensure safe operation.

The design also includes a **crystal oscillator circuit** and **capacitors** required for stable microcontroller operation.

## ⚙ Circuit Components

Main components used in this project:

- AT89C51 Microcontroller
- LEDs
- Resistors (Current limiting)
- Crystal Oscillator
- Capacitors
- Power Supply

## 🔧 Circuit Functionality

- The **AT89C51 microcontroller** controls LED outputs through its I/O pins.
- Each LED is connected via a resistor to prevent excessive current.
- The crystal oscillator provides the clock signal required for the microcontroller.
- Capacitors stabilize the oscillator circuit.

## 🖥 Simulation

The circuit was designed and tested using **Proteus simulation software**.
